# System Monitor Tool

A Simple bash-based system monitoring tool.

## Features
- CPU usage monitoring
- Memory usage tracking
- Disk space alerts
- Log file genaration

## Installation
```
git clone https://github.com/YOUR-USERNAME/system-monitor.git
cd system-monitor
chmod +x monitor.sh
```

## Usage 
./monitor.sh --interval 5
